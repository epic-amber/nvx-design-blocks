# Design system tokens (Tailwind)

Все значения заданы в `tailwind.config.ts`. Hero и остальные блоки используют только эти токены.

## Spacing (отступы)

Шкала Tailwind: **1 unit = 0.25rem (4px)**. Используйте классы `p-*`, `m-*`, `gap-*`, `space-*`.

| Класс   | Значение  | rem    | Пример использования      |
|---------|-----------|--------|---------------------------|
| 1       | 4px       | 0.25rem| gap-1, p-1                |
| 2       | 8px       | 0.5rem | gap-2 (иконка + подпись)  |
| 4       | 16px      | 1rem   | gap-4 (заголовок + блок)  |
| 5       | 20px      | 1.25rem| gap-5 (кнопки mobile), mb-5 |
| 6       | 24px      | 1.5rem | py-6 (блок контента)      |
| 7       | 28px      | 1.75rem| gap-7 (контент hero), кнопки md+ |
| 8       | 32px      | 2rem   | —                         |
| 9       | 36px      | 2.25rem| gap-9 (текст ↔ изображение) |
| 12      | 48px      | 3rem   | py-12 (секция mobile)     |
| 16      | 64px      | 4rem   | py-16 (sm)                |
| 20      | 80px      | 5rem   | py-20 (md)                |
| 24      | 96px      | 6rem   | py-24 (lg)                |
| 32      | 128px     | 8rem   | py-32 (xl секция)         |

**Кастомные:**

| Класс        | Значение   | Использование              |
|--------------|------------|----------------------------|
| `col-gap`    | 4.375rem (70px) | xl: gap между колонками hero |
| `btn`        | 2.875rem (46px) | Высота кнопок (h-btn)       |
| `btn-min`    | 9.375rem (150px)| min-width основной кнопки  |

**Контейнер (горизонтальные отступы):** `px-4` (mobile) → `sm:px-6` → `lg:px-8`.

## Typography (шрифты)

| Класс           | size   | line-height | Использование           |
|-----------------|--------|-------------|-------------------------|
| `text-caption`  | 12px (0.75rem)   | 1.2  | Breadcrumb, мелкие подписи |
| `text-subtitle` | 18px (1.125rem) | 1.2  | Label (например «IoT Query») |
| `text-body-lg`  | 20px (1.25rem)  | 1.4  | Описание (desktop)      |
| `text-display-1`| 60px (3.75rem)  | 1.1  | Hero H1                  |
| `text-button`   | 16px (1rem)     | 1.4  | Текст кнопок             |

**Мобильные:** описание — `text-lg` (18px), с `sm:` — `text-body-lg` (20px).

**Hero H1:** mobile `text-4xl` (36px), с `md:` — `text-display-1` (60px).

**Семейства:** `font-body`, `font-display` (Proxima Nova).

## Colors

**Семантические (текст/UI):**

| Класс                 | Hex      | Использование        |
|-----------------------|----------|----------------------|
| `text-content-primary`   | #101828 | Заголовки, основной текст |
| `text-content-secondary` | #1e2939 | Описание, вторичный текст |
| `text-content-muted`     | #62748e | Breadcrumb, подсказки    |

**Бренд:** `brand-500` (#0084d1) — кнопки, ссылки. Оттенки 50–950 в `tailwind.config`.

**Нейтральные:** `neutral-*` — фон плейсхолдера (`neutral-200`), фон навбара (`neutral-900`) и т.д.

## Hero background

Фон Hero задаётся в компоненте (mesh + градиенты по Figma).

| Параметр | Значение | Описание |
|----------|----------|----------|
| Mesh | rgba(223, 242, 254, 0.52), 40×40px | Сетка #dff2fe 52%, маска затухания сверху/снизу |
| Горизонтальный градиент | см. HeroBlock | mix-blend-multiply |
| Оверлей | см. HeroBlock | Затухание сверху/снизу, низ rgb(184,230,254) → белый |
